## Building AI Systems: Techniques I Specialize In

**Designing intelligent systems**
- Agentic system design (planner–executor–judge patterns)
- Task decomposition & tool orchestration
- Memory vs context tradeoff design

**Making AI reliable & trustworthy**
- LLM evaluation & judging (human + LLM-as-judge)
- Guardrails, safety checks, and graceful fallbacks
- Human-in-the-loop workflows & escalation logic

**Scaling AI in production**
- Prompt engineering (production & regression-safe)
- Cost-aware model selection (latency, quality, token economics)
- AI observability (failure modes, drift, confidence tracking)

**Knowledge & retrieval systems**
- RAG pipeline design
- Retrieval quality tuning & failure analysis
